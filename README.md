# Expense List: Fullstack application (SPA)
**Expense List FS** 💰  
_Aplicación que listara mis gastos con sus correspondientes "acciones" usuales._  
Lista de Gastos    
Calcula saldo  
Orden de gastos por fechas  
...  

#### Backend: Spring Boot
#### Frontend: Angular 11
